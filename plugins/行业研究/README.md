# 行业研究 Skills

行业研究相关的 Claude Code skills，帮助完成市场规模测算、竞争格局分析、创业机会判断等工作。

## Skills

### market-sizing (市场空间测算)

科学的市场规模测算方法：
- **Bottom-Up**：从潜在客户基数出发，层层拆解，每层用产业数据验证
- **Top-Down**：从龙头收入和市场份额反推
- **交叉验证**：差异 <20% 可信，>30% 必须检查假设

使用方式：`/market-sizing <行业/市场名称> [--geo 地域]`

### market-opportunity (市场机会判断)

判断市场对创业公司的机会：
- 增速判断（高速/稳定/成熟/衰退）
- 竞争格局（一家独大/双寡头/多强/分散）
- 护城河分析（7 项评估）
- 创业机会（垂直行业/地域/客群/AI Native）

使用方式：`/market-opportunity <行业/市场名称> [--startup-lens]`
