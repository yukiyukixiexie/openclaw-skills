# HK Realtime Quote - 港股实时行情

免费获取港股实时行情数据，支持新浪财经和腾讯财经两个数据源。

## 特点

- **完全免费**：无需注册，无需 API Key
- **近实时**：数据延迟 < 1分钟
- **双数据源**：新浪财经 + 腾讯财经，互为备份
- **全港股覆盖**：支持所有港股代码

## 使用方式

```bash
# 查询单只股票
python skills/hk_quote.py 02513

# 批量查询
python skills/hk_quote.py 02513 0700 9988 --table

# JSON输出
python skills/hk_quote.py 02513 --json
```

## 返回数据

| 字段 | 说明 |
|------|------|
| price | 现价 |
| change_pct | 涨跌幅(%) |
| volume | 成交量 |
| amount | 成交额 |
| high/low | 最高/最低 |
| high_52w/low_52w | 52周高低 |
