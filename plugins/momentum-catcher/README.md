# Momentum Catcher - 资金动量捕捉框架

港股资金动量捕捉分析框架，用于识别事件驱动机会、确认资金共识、捕捉动量加速期。

## 核心理念

**三阶段动量模型**：
1. **事件触发** → 关注但不入场
2. **资金确认** → 确认后建仓
3. **动量加速** → 持有/加仓 → 动量衰减时退出

**核心原则**：
- 不抄底，买确认（放弃底部10%）
- 不逃顶，卖衰减（放弃顶部10%）
- 多信号叠加确认

## 使用方式

```bash
# 完整分析
python skills/indicators.py --ticker 02513

# 实时行情
python skills/indicators.py --ticker 02513 --realtime

# 回测
python skills/backtest/run_backtest.py --ticker 02513
```

## 信号指标

**动量信号** (5项)：
- 价格突破20日高点
- MACD金叉(零轴上方)
- RSI强势区(60-80)
- 均线多头排列
- 成交量异动(>1.5x)

**退出信号**：
- 量价背离
- RSI超买回落
- MACD死叉
- 跌破关键均线
