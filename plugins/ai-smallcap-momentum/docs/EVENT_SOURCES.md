# 事件驱动信息源挖掘框架

## 目标

从**事后复盘**升级为**前置提醒分析**：
- 不是等股价涨了再找原因
- 而是提前发现催化事件，在启动前介入

---

## 一、政策层信息源（最高优先级）

### 1.1 领导人行程追踪

**为什么重要**：习近平视察某企业/园区 = 最强政策背书信号

**信息源**：

| 来源 | URL | 更新频率 | 获取方式 |
|------|-----|----------|----------|
| 新华网-领导人活动 | news.cn/politics/leaders | 实时 | RSS/爬虫 |
| 人民网-时政 | politics.people.com.cn | 实时 | RSS/爬虫 |
| 中国政府网 | gov.cn | 每日 | RSS |
| 央视新闻 | 微信公众号/App | 实时 | API |

**关键词监控**：
```
习近平 + [考察|视察|调研|座谈|会见]
总书记 + [科技|人工智能|创新|企业]
国家领导人 + [亦庄|中关村|张江|深圳]
```

**搜索示例**：
```bash
# Brave Search
python fetch_events.py --search "习近平 考察 科技企业 2026"
python fetch_events.py --search "总书记 座谈会 人工智能"
```

### 1.2 高层座谈会参与者追踪

**为什么重要**：创始人参与总书记座谈会 = 行业龙头地位确认

**信息源**：

| 来源 | 信息类型 | 获取方式 |
|------|----------|----------|
| 新华社通稿 | 座谈会参与者名单 | 文本解析 |
| 企业官方公众号 | 创始人动态 | 监控 |
| 财经媒体 | 会议解读 | 搜索 |

**追踪逻辑**：
```
1. 监控"总书记座谈会"新闻
2. 提取参与者名单
3. 匹配到上市公司 → 生成信号
4. 关联历史：该创始人是否多次参与？
```

**智谱案例**：
```
唐杰参与记录：
- 2024-05: 总书记座谈会 (首次)
- 2026-02-09: 亦庄视察交流 (第二次)

规律: 多次参与 = 深度认可 = 更强信号
```

### 1.3 政策文件追踪

**信息源**：

| 来源 | URL | 关注内容 |
|------|-----|----------|
| 国务院 | gov.cn/zhengce | 产业政策、规划 |
| 发改委 | ndrc.gov.cn | 项目审批、资金 |
| 工信部 | miit.gov.cn | 行业标准、试点 |
| 科技部 | most.gov.cn | 科研项目、补贴 |

**关键词**：
```
人工智能 + [专项|补贴|试点|示范]
大模型 + [政策|规划|标准]
信创 + [目录|采购|替代]
```

### 2.1 港交所披露易（公告）

**关注公告类型**：
- 须予公布的交易
- 关连交易

### 2.2 公司官方渠道

| 渠道 | 信息类型 | 监控方式 |
|------|----------|----------|
| 官网新闻 | 产品发布、合作 | 爬虫 |
| 微信公众号 | 动态、活动 | 微信搜索 |

### 2.3 创始人/高管动态

**为什么重要**：创始人出现在重要场合 = 公司地位信号

**追踪方式**：
```
1. 创始人姓名 + [演讲|参会|获奖|采访]

3. 高管变动 + 公司名称
```

**智谱案例**：
```
追踪: "唐杰" OR "张鹏" + 智谱
发现: 唐杰参加亦庄交流 → 02-09提前信号
```

**关注点**：

- 采购目录更新
- 补贴政策出台

## 六、前置分析工作流

### 6.1 日常监控清单

```markdown
## 每日监控 (早9点)

### 政策层
- [ ] 新华网领导人活动页
- [ ] 国务院政策发布
- [ ] 重要会议预告

### 公司层
- [ ] 港交所公告 (关注标的)
- [ ] 公司公众号更新
- [ ] 创始人社交媒体

### 行业层
- [ ] 竞品动态
- [ ] 行业新闻

### 资金层
- [ ] 南向资金流向
- [ ] 大宗交易记录
```

### 6.2 事件发现 → 信号生成流程

```
Step 1: 事件发现
├─ 来源: 新闻/公告/社交媒体
└─ 工具: fetch_events.py

Step 2: 事件分类
├─ 政策背书类 (领导人、政策文件)
├─ 产品发布类 (模型、应用)
├─ 商业验证类 (提价、合同、客户)
├─ 资金确认类 (机构入场、增持)
└─ 舆情扩散类 (媒体报道、热搜)

Step 3: 信号强度评估
├─ 政策背书: ⭐⭐⭐⭐⭐ (最强)
├─ 产品发布: ⭐⭐⭐⭐
├─ 商业验证: ⭐⭐⭐
├─ 资金确认: ⭐⭐⭐
└─ 舆情扩散: ⭐⭐

Step 4: 与股价结合
├─ 事件发生 + 股价未动 → 潜在机会
├─ 事件发生 + 股价已动 → 评估追入风险
└─ 无明确事件 + 股价异动 → 挖掘隐藏催化
```

### 6.3 信号输出模板

```markdown
## 事件信号报告

**股票**: {ticker}
**日期**: {date}
**信号强度**: ⭐⭐⭐⭐⭐

### 事件
{event_description}

### 来源
- 新闻: {news_url}
- 公告: {announcement_url}

### 关联分析
- 创始人历史: {founder_history}
- 政策背景: {policy_context}
- 竞品动态: {competitor_news}

### 股价状态
- 当前价: {price}
- 距前高: {distance_to_high}%
- 流通市值: {float_cap}

### 操作建议
{recommendation}
```

## 八、智谱案例复盘（信息源验证）

### 如果在02-09之前做前置分析

```
Step 1: 领导人行程监控
├─ 来源: 新华网
├─ 发现: 2月9日习近平将考察北京
└─ 预判: 可能涉及科技/信创领域

Step 2: 创始人追踪
├─ 来源: 唐杰此前参与过总书记座谈会
├─ 关联: 智谱是信创园区重要企业
└─ 预判: 唐杰可能参与此次交流

Step 3: 提前布局
├─ 时点: 2月6日~8日 (春节后开盘前)
├─ 股价: ~220 港元 (回调后低位)
└─ 信号: 政策催化预期 → 提前入场
```

### 实际结果

```
02-09: 习近平视察亦庄，唐杰参与交流
股价: 214 → 277 (+36.2%)

如果提前布局:
├─ 入场价: ~220
├─ 催化后: 277 (+26%)
└─ 后续高点: 725 (+230%)
```
