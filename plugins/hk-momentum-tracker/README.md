# HK Momentum Tracker

港股 AI 概念股资金阶段追踪与事件驱动交易策略

## 核心交易哲学

**这不是估值钱，不是基本面钱。**

你要赚的是：
```
叙事升级 × 流动性挤压 × 资金拥挤加速
```

**利润来源：**
1. 预期重估
2. 仓位集中
3. 供给锁死
4. 情绪正反馈

**抓的是：趋势中段 → 加速初段 的非线性斜率**

## 大行情的结构

```
突破盘整 → 连续放量 → 回撤极小 → 加速 → 情绪爆发
```

**关键认知：**
- 大行情回调很浅，犹豫就上不去
- 入场点不是"舒服回调"，而是**突破确认后的第一次延续**

## 进攻型入场策略

### 条件一：叙事被市场验证
不是你觉得好，而是市场确认了：
- 放量突破（成交额 > P90）
- 创阶段新高
- 成为市场焦点

### 条件二：回撤被迅速买回
- 单日跌 8%，第二天收回
- = 承接力强，资金在抢筹

### 条件三：突破后第一次延续
- 突破后 1-3 天
- 站稳突破位（> 95%）
- 成交额未萎缩

## 仓位管理

| 阶段 | 仓位 | 操作 | 核心纪律 |
|------|------|------|----------|
| PRE-EVENT | 10-20% | 期权式入场 | 亏损有限 |
| EVENT_DAY | 30-40% | 确认后加仓 | 尾盘加仓 |
| AGGRESSIVE ENTRY | 25-40% | 突破延续入场 | 不等回调 |
| ACCELERATION | 锁定 | **禁止加仓** | 只管理 |
| DISTRIBUTION | 减至20% | 减仓 | 保护利润 |

## 使用方法

```bash
/hk-momentum-tracker 00020           # 综合分析
/hk-momentum-tracker 00020 --history # 上市以来全量复盘
/hk-momentum-tracker 00020 --daily   # 每日追踪
```
