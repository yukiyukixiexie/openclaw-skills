# OpenClaw Skills

OpenClaw 投资分析 Skills 集合

## Skills

### earnings-decoder
买方视角财报解读 - 从价格反推假设，用数据验证，输出仓位决策

### investment-assistant  
投资助手 - 每日投资日报、财报分析、选股建议

## 安装

将 skills 目录复制到 OpenClaw skills 路径：

```bash
cp -r earnings-decoder /root/.openclaw/skills/
cp -r investment-assistant /root/.openclaw/skills/
```
